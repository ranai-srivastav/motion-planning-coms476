Put the following files in the same folder or in your PYTHONPATH
* main.py
* queue.py
* discrete_search.py

Suppose that the problem description is provided in project1_desc.json.

1. For BFS, run

$ python hw1.py grid_desc.json --alg bfs --out soln_bfs.json

which will output project1_bfs.json.

2. For DFS, run

$ python hw1.py grid_desc.json --alg dfs --out soln_dfs.json

which will output project1_dfs.json.

3. For A*, run

$ python hw1.py grid_desc.json --alg astar --out soln_astar.json
